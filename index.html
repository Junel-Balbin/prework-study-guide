HTML

  • Fundamental building block of a web page.
  • Defines the structure and content of a web page.

CSS

  • Style the appearance of the web page.
  • Able to change size, color and font.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch & switches to it.

JavaScript

  • Add interactive elements on a web page.
  • Make webpages more dynamic and much more fun.
  • A variable is a named container that allows us to store in our code.
  • Control flow is the order in which a computer executes code in a script.
